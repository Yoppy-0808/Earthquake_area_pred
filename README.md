# Earthquake_area_pred
## 内　容
  地域別地震確率の予測（10/30日）として、LightGBM手法を利用し、予測モデルの作成を行いました。  
  106年という大規模データをモデル作成に使用し、特徴量を増やすことで調整を行っています。  

## 使用環境
  Google colaboratory

## 使用データ
- 気象庁：震度データベース検索（https://www.data.jma.go.jp/eqdb/data/shindo/）  
  ※出典：気象庁　震度データベース検索  
  API で取得した生データを CSV に変換し、可視化・加工したものを利用しています。  

- 観測地点情報：stations.json  
  ※@iku55さんのGitHubに掲載されているstations.json ファイルより震度観測点情報を取得し、  
  データ加工に利用させていただいてます。  

## 注意事項
- 予測日について  
  デフォルトでのデータ取得は1919/01/01～2025/12/31となっています。  
  必要に応じて、取得する年月日の変更を行って下さい。  
  デフォルトは2025/12/31から10日もしくは30日の予測としています。  
- 使用データについて  
  緯度経度情報から都道府県の割り当てを行うために、stations.jsonを利用しています。  
  上記ファイルをダウンロード後、同じディレクトリ内に格納してから実行を行ってください。  
- 予測について  
  最終表示のあるUIで、都道府県を選択し、「予測する」ボタンを押下して下さい。  
  表示には少し時間がかかります。
